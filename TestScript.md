## Scan the library for significant errors

### Run the test script

Clone the repository

```bash
git clone https://github.com/jackkum/node-pdu.git
```

Enter directory

```bash
cd ./node-pdu
```

Install all dependencies

```bash
npm i
```

Build the library

```bash
npm run build:lib
```

Build the test script

```bash
npm run build:test
```

Run the test script

```bash
node ./test/
```
